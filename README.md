### Prerequisites

- Node.js (v14 or higher)
- npm or yarn package manager

### Installation

1. Clone the repository:
```bash
git clone https://github.com/abdelftah-shkal/task_manger.git
cd task_manger
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```
